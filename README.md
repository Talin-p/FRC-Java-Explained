# FRC.learn — From Zero to Competition Robot

## Overview
FRC.learn is a browser-based interactive learning platform designed to teach complete beginners how to program FIRST Robotics Competition (FRC) robots using Java and WPILib.

The goal is to take a student from **zero programming experience** to being able to understand, build, and modify a command-based FRC robot before competition season.

The application combines:
- Structured learning paths
- Interactive lessons
- Visual diagrams
- Live coding sandboxes
- Robot simulations
- Knowledge checks and quizzes
- Progress tracking and XP systems

---

## Target Audience
- Students with no prior coding experience
- New FRC programmers
- FRC teams onboarding rookie members
- Mentors looking for a guided teaching resource

---

## Learning Path

### Phase 1 — Java Fundamentals
1. Variables
2. If Statements
3. Loops
4. Methods
5. Classes & Objects

### Phase 2 — WPILib & Command-Based Programming
6. Subsystems
7. Commands
8. RobotContainer
9. Autonomous Programming
10. Competition Robot Integration

Each lesson builds directly on previous concepts so students learn programming in the same order they will encounter it while developing a real robot.

## Educational Philosophy

FRC.learn is designed around the principle that students learn best when:

1. Concepts are introduced gradually.
2. Every new idea connects to previous ideas.
3. Visual explanations accompany code.
4. Practice happens immediately after instruction.
5. Examples use real FRC scenarios instead of generic programming problems.

Instead of teaching Java first and robotics later, the platform teaches Java through robotics examples from day one.
